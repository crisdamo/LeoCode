HTML

HTML o Lenguaje de Marcas de Hipertexto (HyperText Markup Language)

Es un lenguaje para programar paginas web

Y para que este programa funcione necesitamos una sintaxis (Lo de la sintaxis te lo explico aquí)

Pero antes necesitamos donde escribir el codigo, asi que vamos a usar el block de notas

en tu Bloc de Notas vas a escribir el codigo de abajo.....

La sintaxis es:

<html>

<head>

<title>Mi Primer Web</title>

<body>

<p>Hola Mundo!<p>

</body>

</html>
